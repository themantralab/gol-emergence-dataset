# Figures

> **Full dataset:** [huggingface.co/datasets/themantralab/gol-emergence-dataset](https://huggingface.co/datasets/themantralab/gol-emergence-dataset)

All figures are generated by `src/generate_data.py` as part of the Stage 1 pipeline.

| File | Description |
|---|---|
| `signal_samples.png` | Individual seed trajectories for 3 seeds per class. Shows what raw signal shapes look like per behavioral class — dying decays, still life is flat, oscillator is periodic, glider drifts. |
| `signal_distributions.png` | Violin plots of all 10 signals split by behavioral class. Verifies inter-class discriminability — P and Δcx/Δcy show clear separation. |
| `tsne_signatures.png` | t-SNE of FFT-magnitude signatures (10k proportional subsample), colored by class. Dying and glider are clearly separated; gliders cluster with oscillators in FFT space, which is the known limitation that motivated the z-cloud LOF design in Stage 3. |

Additional analysis figures will be added in future releases as later stages are completed.
