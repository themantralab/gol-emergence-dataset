# Figures

> **Full dataset:** [huggingface.co/datasets/themantralab/gol-emergence-dataset](https://huggingface.co/datasets/themantralab/gol-emergence-dataset)

All figures are generated by `src/analyse_data.py` from the Stage 1 dataset.

| File | Type | Description |
|---|---|---|
| `signal_samples.png` | Samples | Individual seed trajectories (3 per class). Shows raw signal shapes per behavioral class. |
| `signal_distributions.png` | Characterization | Violin plots of all 10 signals split by class. Verifies inter-class discriminability. |
| `tsne_signatures.png` | Geometry | t-SNE of FFT-magnitude signatures (10k subsample, proportional). Shows glider/dying separation and the FFT limitation that motivated z-cloud LOF. |
| `lifespan_hist.png` | Characterization | Lifespan distribution before and after stratification. Confirms bucket weighting worked. |
| `normalization_check.txt` | Verification | Raw vs normalized signal densities. Confirms mean≈0 post-normalization. |
| `cluster_summary.png` | Clustering | Cluster size bar chart with dominant class label. Verifies behavioral alignment. |
| `temporal_glider.png` | Samples | Temporal trajectory of a representative glider seed. |
| `temporal_oscillator.png` | Samples | Temporal trajectory of a representative oscillator seed. |

Additional analysis figures (PCA, k-NN distance distributions) will be added in future releases.
